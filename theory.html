Theory of Differential Equations

Comprehensive guide to exact and non-exact differential equations

Exact Differential Equations

Definition

A differential equation of the form:

M(x,y)dx + N(x,y)dy = 0

is said to be exact if there exists a function u(x,y) such that:

∂u/∂x = M(x,y) and ∂u/∂y = N(x,y)

Condition for Exactness

The necessary and sufficient condition for the differential equation to be exact is:

∂M/∂y = ∂N/∂x

This condition ensures that the mixed partial derivatives are equal, which is required by Clairaut's theorem.

Method of Solution

  1. Step 1: Verify Exactness

    Check if ∂M/∂y = ∂N/∂x

  2. Step 2: Integrate M with respect to x

    u(x,y) = ∫M(x,y)dx + φ(y)

  3. Step 3: Differentiate with respect to y

    ∂u/∂y = ∂/∂y[∫M(x,y)dx] + φ'(y)

  4. Step 4: Compare with N(x,y)

    Set ∂u/∂y = N(x,y) and solve for φ'(y)

  5. Step 5: Integrate to find φ(y)

    φ(y) = ∫φ'(y)dy

  6. Step 6: Write the general solution

    u(x,y) = C, where C is the constant of integration

Example

Solve: (2xy + y²)dx + (x² + 2xy)dy = 0

Step 1: Check exactness

M = 2xy + y², N = x² + 2xy

∂M/∂y = 2x + 2y

∂N/∂x = 2x + 2y

Since ∂M/∂y = ∂N/∂x, the equation is exact.

Step 2: Integrate M with respect to x

u(x,y) = ∫(2xy + y²)dx = x²y + xy² + φ(y)

Step 3: Differentiate with respect to y

∂u/∂y = x² + 2xy + φ'(y)

Step 4: Compare with N

Set x² + 2xy + φ'(y) = x² + 2xy

Thus, φ'(y) = 0

Step 5: Integrate φ'(y)

φ(y) = C₁

Step 6: General solution

u(x,y) = x²y + xy² = C

Non-Exact Differential Equations

Definition

A differential equation of the form:

M(x,y)dx + N(x,y)dy = 0

is said to be non-exact if:

∂M/∂y ≠ ∂N/∂x

Integrating Factors

Non-exact equations can sometimes be made exact by multiplying by an integrating factor μ(x,y):

μ(x,y)M(x,y)dx + μ(x,y)N(x,y)dy = 0

The new equation becomes exact if μ satisfies:

∂(μM)/∂y = ∂(μN)/∂x

Finding Integrating Factors

Case 1: Integrating factor depends only on x

If (∂M/∂y - ∂N/∂x)/N is a function of x only, say f(x), then:

μ(x) = e^∫f(x)dx

Case 2: Integrating factor depends only on y

If (∂N/∂x - ∂M/∂y)/M is a function of y only, say g(y), then:

μ(y) = e^∫g(y)dy

Case 3: Special integrating factors

For equations of specific forms, we can try:

  • μ = x^m y^n
  • μ = e^(αx + βy)
  • μ = 1/(xM + yN) for homogeneous equations

Example

Solve: (3x²y + 2xy + y³)dx + (x² + y²)dy = 0

Step 1: Check exactness

M = 3x²y + 2xy + y³, N = x² + y²

∂M/∂y = 3x² + 2x + 3y²

∂N/∂x = 2x

Since ∂M/∂y ≠ ∂N/∂x, the equation is non-exact.

Step 2: Try integrating factor

(∂M/∂y - ∂N/∂x)/N = (3x² + 2x + 3y² - 2x)/(x² + y²) = 3

This is a constant, so we can use μ = e^∫3dx = e^(3x)

Step 3: Multiply by integrating factor

The equation becomes exact and can be solved.

Solution Methods

Direct Integration

For separable equations where variables can be separated.

dy/dx = f(x)g(y)

Homogeneous Equations

Equations where M and N are homogeneous functions of the same degree.

Use substitution: y = vx

Linear Equations

Equations of the form dy/dx + P(x)y = Q(x).

Integrating factor: e^∫P(x)dx

Exact Equations

Solve by finding potential function u(x,y).

∂M/∂y = ∂N/∂x

Real-World Example: Population Growth

The logistic growth model is described by:

dP/dt = rP(1 - P/K)

Where:

  • P = Population size
  • r = Intrinsic growth rate
  • K = Carrying capacity

This non-linear differential equation models how populations grow with limited resources.
