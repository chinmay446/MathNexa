Exact Differential Equations

Fundamental Concept

Definition

A first-order differential equation of the form:

M(x, y)dx + N(x, y)dy = 0

is called exact if there exists a function ψ(x, y) such that:

dψ = M dx + N dy

Equivalently, the equation is exact if and only if:

\frac{\partial M}{\partial y} = \frac{\partial N}{\partial x}

Why This Condition?

If such a function ψ(x, y) exists, then by the equality of mixed partial derivatives:

\frac{\partial^2 \psi}{\partial y \partial x} = \frac{\partial^2 \psi}{\partial x \partial y}

Since:

\frac{\partial \psi}{\partial x} = M \quad \text{and} \quad \frac{\partial \psi}{\partial y} = N

We have:

\frac{\partial M}{\partial y} = \frac{\partial^2 \psi}{\partial y \partial x} = \frac{\partial^2 \psi}{\partial x \partial y} = \frac{\partial N}{\partial x}

This condition is both necessary and sufficient for exactness (on a simply connected domain).

Solution Method for Exact Equations

If the equation is exact, the solution ψ(x, y) = C (where C is a constant) can be found by:

  1. Integrate M with respect to x, treating y as constant:
    ψ(x, y) = ∫ M dx + g(y)
  2. Differentiate this result with respect to y and set equal to N:
    \frac{\partial ψ}{\partial y} = \frac{\partial}{\partial y} ∫ M dx + g'(y) = N
  3. Solve for g'(y) and integrate to find g(y)
  4. The general solution is ψ(x, y) = C

Example: Exact Equation

Consider the equation:

(2xy) dx + (x^2) dy = 0

Here, M = 2xy and N = x².

Check exactness:

\frac{\partial M}{\partial y} = 2x, \quad \frac{\partial N}{\partial x} = 2x

Since they are equal, the equation is exact.

Solution:

ψ(x, y) = ∫ 2xy dx = x^2 y + g(y)
\frac{\partial ψ}{\partial y} = x^2 + g'(y) = N = x^2
g'(y) = 0 \Rightarrow g(y) = C_1
ψ(x, y) = x^2 y = C

Integrating Factors

Making Non-Exact Equations Solvable

What is an Integrating Factor?

If a differential equation M dx + N dy = 0 is not exact, we can sometimes multiply it by a function μ(x, y) to make it exact. Such a function is called an integrating factor.

After multiplication by μ, the equation becomes:

μM dx + μN dy = 0

which is exact if:

\frac{\partial (μM)}{\partial y} = \frac{\partial (μN)}{\partial x}

Finding Integrating Factors

There are special cases where we can find integrating factors easily:

Case 1: μ depends only on x

If (∂M/∂y - ∂N/∂x)/N is a function of x only, say f(x), then:

μ(x) = e^{\int f(x) dx}

where:

f(x) = \frac{\frac{\partial M}{\partial y} - \frac{\partial N}{\partial x}}{N}

Case 2: μ depends only on y

If (∂N/∂x - ∂M/∂y)/M is a function of y only, say g(y), then:

μ(y) = e^{\int g(y) dy}

where:

g(y) = \frac{\frac{\partial N}{\partial x} - \frac{\partial M}{\partial y}}{M}

Case 3: Special Forms

For equations of specific forms, we can use known integrating factors:

  • If M and N are homogeneous functions of the same degree: μ = 1/(xM + yN)
  • If the equation is of the form y f(xy) dx + x g(xy) dy = 0: μ = 1/(xy(M - N))

Example: Using Integrating Factor

Consider the non-exact equation:

y dx + 2x dy = 0

Here, M = y, N = 2x.

Check exactness:

\frac{\partial M}{\partial y} = 1, \quad \frac{\partial N}{\partial x} = 2

Not equal, so the equation is not exact.

Try to find μ(x):

\frac{\frac{\partial M}{\partial y} - \frac{\partial N}{\partial x}}{N} = \frac{1 - 2}{2x} = -\frac{1}{2x}

This is a function of x only, so:

μ(x) = e^{\int -\frac{1}{2x} dx} = e^{-\frac{1}{2} \ln x} = x^{-1/2}

Multiply the equation by μ:

x^{-1/2} y dx + 2x^{1/2} dy = 0

Now check exactness of the new equation...

Solution Methods

Step-by-Step Procedures
1

Check Exactness

Compute ∂M/∂y and ∂N/∂x. If equal, proceed to solve as exact equation.

∂M/∂y = ∂N/∂x
2

Find Integrating Factor

If not exact, check if (∂M/∂y - ∂N/∂x)/N depends only on x, or if (∂N/∂x - ∂M/∂y)/M depends only on y.

3

Multiply and Solve

Multiply the equation by the integrating factor to make it exact, then solve as an exact equation.

4

Verify Solution

Differentiate the solution implicitly to verify it satisfies the original differential equation.

Start with M dx + N dy = 0

↓

↓ YES

Equation is exact

Solve by finding ψ(x,y) such that dψ = M dx + N dy

↓ NO

Find integrating factor

Check for μ(x) or μ(y)

↓

Solution: ψ(x,y) = C

Worked Examples

Practice Problems with Solutions

Exact Equation: (3x² + y) dx + (x - 2y) dy = 0

Step 1: Identify M and N

M = 3x² + y, N = x - 2y

Step 2: Check exactness

∂M/∂y = 1, ∂N/∂x = 1

Since they are equal, the equation is exact.

Step 3: Find ψ(x, y)

ψ = ∫ M dx = ∫ (3x² + y) dx = x³ + xy + g(y)

Step 4: Differentiate and compare with N

∂ψ/∂y = x + g'(y) = N = x - 2y
g'(y) = -2y ⇒ g(y) = -y²

Step 5: Write general solution

ψ(x, y) = x³ + xy - y² = C

Non-Exact Equation: y dx - x dy = 0

Step 1: Identify M and N

M = y, N = -x

Step 2: Check exactness

∂M/∂y = 1, ∂N/∂x = -1

Not equal, so the equation is not exact.

Step 3: Find integrating factor

\frac{∂M/∂y - ∂N/∂x}{N} = \frac{1 - (-1)}{-x} = -\frac{2}{x}

This is a function of x only, so:

μ(x) = e^{\int -2/x dx} = e^{-2\ln x} = x^{-2}

Step 4: Multiply and solve

After multiplying by μ = 1/x², the equation becomes exact and can be solved.

Exact Equation: (2xy + y²) dx + (x² + 2xy) dy = 0

Step 1: Identify M and N

M = 2xy + y², N = x² + 2xy

Step 2: Check exactness

∂M/∂y = 2x + 2y, ∂N/∂x = 2x + 2y

Equal, so exact.

Step 3: Solution

ψ(x, y) = x²y + xy² = C

Applications

Real-World Uses

Physics

Newton's laws of motion, heat transfer, wave equations, and quantum mechanics all involve differential equations.

Engineering

Circuit analysis, control systems, fluid dynamics, and structural analysis rely on differential equations.

Economics

Economic growth models, option pricing, and market equilibrium analysis use differential equations.

Biology

Population dynamics, epidemiology, and biochemical reactions are modeled with differential equations.
